HTML-based Combinatorial Games

These are implemented in HTML so they can be played easily on any web-enabled device. (You will need to allow JavaScript.)

Here are the games that are working right now:

  • Atropos
  • Clobber
  • Domineering
  • Clobbineering
  • Anti-Clobber
  • Buttons and Scissors
  • NoCanDo
  • Connect Four
  • Manalath

Here is the GitHub repo for this project. (There are a lot of improvements that can be made.)
